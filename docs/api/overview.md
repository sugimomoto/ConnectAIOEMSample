# API概要

## 目的と機能

CData Connect AI APIは、設定済みデータソースへのアクセスを実現するために設計されています。CDataは、人気のある統合をデータソースに接続するためのカスタムプラグインとドライバを提供しており、カスタムソリューションが存在しない場合には標準エンドポイントを提供しています。

## 主要なAPI

**REST API** が中心的な役割を担っています。このインターフェースは、HTTPリクエストを作成できるすべてのREST互換アプリケーションまたは統合ツールに対応しており、以下の機能をサポートしています：

- データクエリ実行
- バッチクエリ操作
- ストアドプロシージャ実行

## アーキテクチャの特徴

- **JSON形式** に基づいたAPI
- 複数のデータソースへの**ユニファイドアクセス**を実現
- 別途のドライバやプラグインがない場合でも、統一されたエンドポイントを通じてデータと連携可能

## ベースURL

```
https://cloud.cdata.com/api
```
